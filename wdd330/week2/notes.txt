Create a Next Tech `Browser` tab by clicking the `+` and select Browser. Open the html file to view it.
Click the refresh button if your changes do not reflect.

To access the DOM of your html file with the JavaScript console:
1. Open it in a new tab by clicking the `Open in Browser Tab` (right side of the address bar).
2. Press F12 (or Command + Option + I on a Mac) to open the Developer Tools.
3. Click on the Console tab.
